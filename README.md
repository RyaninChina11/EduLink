# EduLink智能班级中枢

## 📋 现有问题和解决方向
### ⚠️现有问题：
1.上课铃响后，学生不知道上什么课，无法及时做好课前准备
2.师生不知道当前天气，无法及时合理安排户外活动
3.老师要跟学生转达信息（如：班务信息，课堂作业，默写信息等）时要在教室和办公室之间奔走，比较耽误时间和效率
### ✅解决方向：
1.在预备铃响后，本作品自动播报接下来的课程，并显示在屏幕上
2.通过本作品查询天气，每隔一小时更新一次天气，并显示在屏幕上
3.通过从本团队开发的应用程序发送消息，本作品接收并播报消息
